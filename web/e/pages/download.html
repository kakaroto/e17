DR16 Downloads

The Enlightenment DR0.16.5 release has been considered stable since October 27th, 2000. Before downloading, review the following list of required libraries and applications:

NOTE: ftp.enlightenment.org no longer exists! Please go here instead.

  • IMLIB v1.9.14
    • libTIFF 3.5.5
    • libPNG 1.0.8
    • Zlib 1.1.3
    • libJPEG 6b
    • libUNGIF 4.1.0 (optional)
    • GTK+ 1.2.1 (optional)
    • GLIB 1.2.1 (optional)
  • FNLIB 0.5 (or greater)
  • FreeType 1.3.1 (or greater)

Enlightenment can be compiled on a number of different *NIX platforms. Below are packages of the current, stable release of Enlightenment. Please choose from the list of available distributions the one that is right for you.

Distribution/Platform File
Enlightenment Source Code TAR.GZ enlightenment-0.16.5.tar.gz
Source Package RPM enlightenment-0.16.5-1.src.rpm
Linux i386 Binary RPM enlightenment-0.16.5-1.i386.rpm
Debian GNU/Linux DEB Packages for 'stable', 'testing' and 'unstable'
Solaris 2.6 (or higher) SPARC packages E DR16.5 + All Requirements
AIX 4.3.3/5L RPMs E DR16.4 + All Requirements from IBM
IRIX 6.2 (or higher) TARDISTs E DR16.4 + All Requirements
GNU Darwin E DR16.5 + All Requirements
Linux/PPC Binary RPM Enlightenment-0.17 for linuxppc

For those interested, E is available directly from CVS. Find out more on the Source Code page.

If you are interested in aquiring releases IMLIB2, EVAS, EDB and more, please look at the files section of our SourceForge Page.
