int evas_get_colors(Evas e);
This function returns the maximum colour allocation count for the Evas passed in as an
argument. This does not mean it is the actual number of colours allocated for use, but the
maximum number allowed to be allocated.
Return:
The return value is an integer in a range from 2 to 256 which is the maximum number of
colours to be allocated when displaying on a display of depth less than or equal to 8 bits.
Arguments:
e
A valid Evas canvas handle.
