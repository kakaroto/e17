CVS Development Sources

All of Enlightenment's development occurs in CVS. You can pull the code directly from CVS using the following commands: 

cvs -d:pserver:anonymous@@cvs.enlightenment.sourceforge.net:/cvsroot/enlightenment login
cvs -z3 -d:pserver:anonymous@@cvs.enlightenment.sourceforge.net:/cvsroot/enlightenment co modulename

Use the following as module names:

Module nameContent
e17[E17's components] [Build Notes - READ THIS!]
e16E 16's sources
etermEterm's sources
miscMiscellaneous tools
webThe website

For those without direct access to CVS (due to company firewalls or whatever) there are some other options for fetching the source code:

Browse the CVS repository online

http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/enlightenment/

The entire CVS repository (including version history) is available and can be browsed via the web. Admire the pretty colours and syntax highlighting.

Unofficial E-CVS module tarballs

http://ftp.neep.com.au/pub/snapshots/cvs.enlightenment.sourceforge.net/ (HTTP)

ftp://ftp.neep.com.au/pub/neep/pub/snapshots/cvs.enlightenment.sourceforge.net (FTP)

Weekly snapshots of all the modules are available from Neep Consulting's web and FTP sites. Andrew tries to keep the collection vaguely up to date. As of August '01 the tarballs total about 36MB. Contact Andrew Shugg via the "Team" page if you have any enquiries or problems with these; this is not a supported service by the E developers.

SourceForge CVS repository tarball

http://cvs.sourceforge.net/cvstarballs/enlightenment-cvsroot.tar.gz

Warning: this is SourceForge's nightly tarball (or daily snapshot - the nomenclature is inconsistent) of the entire repository. That means full CVS history on every file, Attic, the lot; it's a complete backup. As of August '01 the enlightenment-cvsroot.tar.gz file weighs in at 200MB and is only going to get bigger with time. Not recommended for the faint of heart.

If you download this and create your own local E CVS repository you would be advised to hack up some procmail script to grab diffs from the enlightenment-cvs mailing list as the main repository is updated. For new files and directories you'll probably have to manually get them using the web interface below. Again, this course of action is not for the faint of heart. =)

Our CVS repository is hosted by the good boys and girls at SourceForge. You can get more info about using their CVS servers in their SourceForge CVS HOWTO.

Please note that E 0.17 is intended for use only by developers and those who have the time and programming skill to help out. If you cannot get it to build and run on your own, please do not ask the developers for help. Simply wait until it is released and ready for public consumption.
