Enlightenment DR17


The upcoming release 0.17 of Enlightenment is still under development. At the moment it is a fully operational basic window manager without most of the features that the stable 0.16.5 release provides, and the beginnings of the file manager. It uses most of the components listed on the left -- the file system handling is already done by Efsd, and that the graphics backend is entirely handled by Imlib2 and Evas.

Come back to this page again to stay informed about E 17's current features. If you wish to test E 17 right now, please have a look at the CVS code and build it yourself.
