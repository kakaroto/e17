Welcome to the Enlightenment Project!

Enlightenment is a completely themeable, highly configurable Window Manager for the X Window System, traditionally used in Unix environments.

The current, stable release of Enlightenment is 0.16.5. A feature list is available here, documentation is here.

The development effort for the 0.17 series is well underway and a lot larger than for the 0.16 series, mainly because Enlightenment is going to become a desktop shell. See the FAQs to find out what this means and what not. The subsystems created along the way for purposes of graphics handling, filesystem access etc are organized in standalone libraries that can be used by other projects as well.

Therfore, this site includes information about the Enlightenment Canvas Library (Evas), the imaging library Imlib2, the Enlightenment File System Daemon (EFSD), and the other components of the Enlightenment desktop shell.

If you want to jump right in, you can download everything you need here.

Currently, for support, you have three options: using irc (preferred; #e on irc.openprojects.net), our mailing lists, or email (use email addresses from the team page, but don't bug Raster if you don't need to). Suggestions are welcome.
